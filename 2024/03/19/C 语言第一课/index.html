C语言

C语言第一天

关于变量：

#include<stdio.h>

int main() {
    printf("%dbite\n",sizeof(char));
    printf("%dbite\n",sizeof(int));
    printf("%dbite\n",sizeof(short));
    printf("%dbite\n",sizeof(long));
    printf("%dbite\n",sizeof(long long));
    printf("%dbite\n",sizeof(float));
    printf("%dbite\n",sizeof(double));
    return 0;
}
// 显示变量容器的大小
  • 上面是变量容器的大小，以比特字符容量为基础。
  • return 0；相当于停止读取以下代码，在将printf放在最下面时，会出现，后一行无法运行。
  • 在MySQL中一个字符是俩比特，这个不知道是不是具体得后面查找。
  • 毫无疑问的是它的命名并不像python那么自由，需要相应的定义，但隐隐约约觉得他的内存利用率很高，这可能是他的有点，像个刻刀一步一步的往下雕刻。
  • 另外C语言里的大括号有点像一个存放各种仪器的仓库，现在学习的printf，以及scanf这些都得在仓库里面运行。
  • 而括号外是全局，这里值得注意的是，可以看成一个属地户籍。比如我在湖北，相当于最外层大括号，依次缩小自己的属地，而我可以调用我上一级的信息地，我可以说我是湖北的，当更近一级的地方出现时，这个数据便被更改的更明确了，因此局部的变量可以被下一级更改后使用。
  • python 中，在对定义函数时，有局部和全局变量需要使用global，globals nonelocal（用了工具容易手残不记得了）这些定义一遍局部变量，或者全局变量才能更改数据。

个人感觉C跟python里面的相似点不止一处，只不过python更暴力
